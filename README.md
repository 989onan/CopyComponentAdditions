# CopyComponentAdditions

A [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader) mod for [Resonite](https://resonite.com/) A Resonite mod that adds a few different copy component additions. currently:
-  adds a copy linked dynamic bone chain, which copies a dynamic bone chain's values to a new dynamic bone chain via copy value components with write back

## Installation
1. Install [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader).
1. Place [CopyComponentAdditions.dll](https://github.com/989onan/AnimatorAdditions/releases/latest/download/CopyComponentAdditions.dll) into your `rml_mods` folder. This folder should be at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\rml_mods` for a default install. You can create it if it's missing, or if you launch the game once with ResoniteModLoader installed it will create the folder for you.
1. Start the game. If you want to verify that the mod is working you can check your Resonite logs.
